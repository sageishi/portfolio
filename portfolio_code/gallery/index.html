Gallery

下記のGalleryでは３つのjQueryプラグインを使用している。
最初のばらまいたように表示する為のプラグインはデベロッパーが提供を中止していて、
GitHub上に別の方が残していたコピーを見つけました。
ただし、見つけたファイルはデータが壊れていて正しく表示しませんでした。
自分でプラグインを修正して、動作するようにしました。
次にMixitupとFancyboxを先程のプラグインと組み合わせました。
動作はしましたが、Mixitupでフィルタリングした状態でもFancyboxのギャラリービューでは
フィルターされず、全てが表示されました。jQueryを追加して、フィルタリングが利くようにしました。
